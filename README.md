# Demo Task Project

📌 Overview
This project contains a full-stack application with:
- Frontend
- Backend

🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript, Angular (TypeScript-based framework)
- Backend: Node.js / Express 
- Database: Mysql

📂 Project Structure
demo_task/
│── frontend/
│── backend/
│── README.md

🚀 How to Run
1. Clone the repository
git clone https://github.com/your-username/demo_task.git

2. Go to project folder
cd demo_task

3. Run Backend
cd backend
npm install
npm start

4. Run Frontend
cd frontend
npm install
npm start

✨ Features
- Angular-based responsive frontend
- RESTful API using Node.js & Express
- Product & Category management
- Modular backend structure (controllers & routes)
